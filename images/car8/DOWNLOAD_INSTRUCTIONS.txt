# Download Instructions for Chevrolet Malibu

## Google Drive Link:
https://drive.google.com/drive/folders/1QrIeum3tr8F73TqlI3F8bt9j7Wp3exud

## Steps:
1. Open the link above in your browser
2. Download all photos from the folder
3. Place them in folder: images\car8
4. Rename main photo as 'main.jpg'
5. Name other photos as '2.jpg', '3.jpg', etc.

## File Structure:
- main.jpg (main photo)
- 2.jpg, 3.jpg, 4.jpg... (additional photos)

## After downloading:
- Update script.js to display real photos
- Check that all images load correctly
