# Новый проект

Стартовый шаблон фронтенд-проекта:

- `index.html` — HTML-каркас страницы
- `styles.css` — базовые стили и цветовая схема
- `script.js` — пример инициализации логики

## Быстрый старт

1. Откройте `index.html` в браузере.
2. Обновите содержимое секций под задачу.
3. Для сборки/деплоя добавьте необходимые инструменты.
